| f(n)             | g(n)              |
+------------------+-------------------+
| 10n              | n^2 - 10n         |    --> 10n = O(n^2 - 10n)
+------------------+-------------------+
| n^3              | n^2 log n         |    --> n^3 = Omega(n^2 log n)
+------------------+-------------------+
| n log n          | n + log n         |    --> n log n  =  Omega(n + log n)
+------------------+-------------------+
| log n            | n^(1/k)           |    --> log n = O(n^(1/k))
+------------------+-------------------+
| ln n             | log n             |    --> ln n = Theta(log n)
+------------------+-------------------+
| log(n + 1)       | log n             |    --> log(n + 1) = Theta(log n), because  log(n + 1) = log(n) + log(1 + 1/n)  --> log(n), n--> infinity
+------------------+-------------------+
| log log n        | log n             |    --> log log n = O(log n)
+------------------+-------------------+
| 2^n              | 10^n              |    --> 2^n = O(10^n)
+------------------+-------------------+
| n^m              | m^n               |    m is fixed  --> n^m = O(m^n)
+------------------+-------------------+
| cos(n * pi / 2)  | sin(n * pi / 2)   |    neither
+------------------+-------------------+
| n^2              | (n * cos n)^2     |    --> n^2 = Omega((n * cos n)^2)
