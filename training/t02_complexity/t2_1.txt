Нехай f(n) = 3n^2 - n + 4. Користуючись означенням покажіть що

a)  f(n) = O(n^2)

exist c and n_0: f(n) <= c*n^2, n >= n_0

c = 3:

3n^2 - n + 4 <= 3n^2
-n + 4 <= 0
n >= 4   --> n_0 = 4

So ==> f(n) = O(n^2)

b) f(n) = Omega(n^2)

exists c and n_0: f(n) >= c*n^2, n >= n_0

c = 2:

3n^2 - n + 4 >= 2n^2

n^2 - n + 4 >= 0

n^2 - n + 1/4 + 15/4 >= 0
(n - 1/2)^2 + 15/4 >= 0  --> true for all n>=0

So ==> f(n) = Omega(n^2)

From a) + b) ==> f(n) = Theta(n^2)
